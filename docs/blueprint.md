# **App Name**: AllianceHub

## Core Features:

- Homepage Display & Navigation: Displays the full responsive homepage with a sticky header, logo, main navigation, highlighted 'Member Login' button, and a mobile hamburger menu. Includes all specified sections like Hero, About Preview, Latest News, Upcoming Events, Members, Gallery Preview, Documents, and Contact CTA, ending with a comprehensive footer.
- Static Content Pages: Provides dedicated pages for 'About Us' (history, mission/vision, leadership), 'Contact' (form, address, map, phone/email), and 'Documents' (list with download/viewer options).
- Dynamic Content Listing & Detail Pages: Manages and displays 'News & Updates' (blog-style with filters/pagination), 'Events' (upcoming/past with details, venue, gallery), 'Gallery' (masonry grid, event albums, lightbox), and a searchable 'Members Directory'.
- User Authentication: Provides a secure member login portal with email/member ID, password fields, and a 'Forgot password' function.
- Member Dashboard Access: After successful login, displays a personalized dashboard showing member's profile, membership details, access to documents, events, and announcements.
- Content Summarization Tool: Automatically generates concise summaries for news articles and event descriptions from uploaded content, assisting content managers in populating the site efficiently.
- Contact Form Processing: Captures and processes submissions from the 'Contact' page form, including name, email, phone, and message, and stores them for administrative review.

## Style Guidelines:

- Primary color: A rich, trustworthy blue (#2259C3) that evokes professionalism and stability.
- Background color: A subtle, desaturated light blue-grey (#EAECE1) to maintain a clean aesthetic while complementing the primary blue.
- Accent color: A radiant gold (#EEBD2B) used sparingly for highlights, call-to-action elements, and premium touches, providing contrast and elegance.
- Headlines font: 'Poppins' (sans-serif) for a modern, precise, and sophisticated heading style.
- Body font: 'Inter' (sans-serif) for its highly legible, objective, and neutral qualities, ensuring readability across all text blocks.
- Utilize a consistent set of modern, minimalist line icons across all sections to visually reinforce content categories and enhance clarity.
- Employ a responsive grid system and card-based designs with soft shadows and rounded corners for a clean, modern aesthetic. The header will be sticky, full-width banners will feature prominently, and galleries will utilize a masonry layout with lightbox previews for images.
- Implement subtle and smooth hover animations on all interactive elements (buttons, navigation links, cards, gallery items) to provide gentle visual feedback and improve user experience without distraction.
